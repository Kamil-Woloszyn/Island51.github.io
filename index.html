Project Background

The project aimed to enhance our Games Design and Development skills by building a group game, following the "Start with Nothing" theme using C++ and SFML. We chose a top-down 2D survival horror game.

We assigned roles, including stakeholder, architect, analyst, and developer, with a project manager overseeing progress. Agile methodology was chosen for flexibility, using GitHub for task organization.

Weekly meetings ensured continuous progress and task assignment. Time constraints emphasized the importance of weekly accomplishments. All meetings were documented and submitted for the assignment.

Project instructions

The player can move around using the WASD keys and aim using the mouse. The player can hit/shoot their weapon using the left mouse button.

The player can switch which weapon they have equipped using the numbers 1 to 5. The player can open their inventory using TAB and open crafting using Q.

In order to play tyhe game the player must save an SFML 2.6.0 File inside of the C:\ directory of their computer
